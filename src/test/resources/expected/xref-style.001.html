1. Customize Individual Cross References

Single cross references can be customized with the xrefstyle Attribute. You can use the %c, %l and %p values from Table 4.1 in the xslTNG Reference Guide. Beware the %p makes sense for PDF output only, since there are no page numbers in HTML. There is also an additional %label value, which stands for the full Label, that is the Component`s name and Number for the targeted Element. The use of these percent-values is explained in the following table, where all cross references target the next Section called Syntax highlighting.

Value of xref/@xrefstyle

Result

Remark
