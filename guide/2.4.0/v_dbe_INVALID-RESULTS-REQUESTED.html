$err:INVALID-RESULTS-REQUESTED — Invalid result form parameter.

Variable:
{http://docbook.org/ns/docbook/errors}INVALID-RESULTS-REQUESTED
Defined in:
modules/errors.xsl
Used in:
docbook.xsl
Used by:
t:docbook

Synopsis

$dbe:INVALID-RESULTS-REQUESTED := xs:QName('dbe:INVALID-RESULTS-REQUESTED')

Description

This error is raised if the $result parameter passed to t:docbook is not one of “raw-results”, “chunked-results”, or “main-document”.
