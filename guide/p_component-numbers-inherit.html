$component-numbers-inherit — Include component labels in section labels?

Parameter:
{}component-numbers-inherit
Defined in:
param.xsl
Used in:
param.xsl, modules/titles.xsl

Synopsis

  |$component-numbers-inherit as xs:string := 'false'

Description

If true, section labels will be prefixed with the label of their containing component (chapter, appendix, etc.) In other words, the first section of chapter 2 will be labeled “2.1” if this is true and simply “1” if it’s false.

See also $division-numbers-inherit, $section-numbers-inherit, and $v:title-properties.
