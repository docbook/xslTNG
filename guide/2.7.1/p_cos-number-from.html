$cos-number-from — Controls numeration of co elements.

Parameter:
{}cos-number-from
Defined in:
param.xsl
Used in:
main.xsl, param.xsl

Synopsis

$cos-number-from as xs:string := 'verbatim section'

Description

Numeration of co elements starts from the most recent node in any of the following categories:

books

See Section 3.6

components

See Section 3.6

sections

See Section 3.6

verbatims

Despite its name, this category incudes not only computeroutput, literallayout, programlisting, screen, imageobjectco, programlistingco and screenco, but also classsynopsis, cmdsynopsis, constructorsynopsis, destructorsynopsis, enumsynopsis, fieldsynopsis, funcsynopsis, macrosynopsis, methodsynopsis, packagesynopsis, typedefsynopsis, unionsynopsis and synopsis.

With the default value, callout numbers will be counted separately for every verbatim or synopsis element, if any (with sections as a fallback). If you don't want a reset for callouts numeration in the whole book, set this parameter to books.
