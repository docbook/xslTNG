m:revhistory-list — Mode to generate a “list” presentation of revhistory.

Mode:
{http://docbook.org/ns/docbook/modes}revhistory-list
Defined in:
modules/blocks.xsl

Description

The revhistory element can be rendered as a list or a table (see $revhistory-style). The m:revhistory-list mode is used to render it as a list.
