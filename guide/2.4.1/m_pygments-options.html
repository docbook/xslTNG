m:pygments-options

m:pygments-options — Calculate the Pygments options for an element.

Mode:
{http://docbook.org/ns/docbook/modes}pygments-options
Defined in:
modules/verbatim.xsl

Description

When the Pygments syntax highlighter is being used, every element that will be highlighted is processed in this mode to generate options for the Pygments process.
