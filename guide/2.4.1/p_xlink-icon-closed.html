$xlink-icon-closed

$xlink-icon-closed — Icon indicating hidden XLink links.

Parameter:
{}xlink-icon-closed
Defined in:
param.xsl
Used in:
main.xsl, param.xsl

Synopsis

  |$xlink-icon-closed := ()

Description

If the JavaScript presentation of extended XLinks is used, this icon marks the place where a multi-ended link can be revealed. The default icon is ▶ (&#x25B6;). See also: $xlink-icon-open.
