$nominal-page-width

$nominal-page-width — The nominal page width.

Parameter:
{}nominal-page-width
Defined in:
param.xsl
Used in:
param.xsl, modules/variable.xsl
Used by:
$image-nominal-width, $v:nominal-page-width

Synopsis

  |$nominal-page-width := '6in'

Description

To calculate the width of the columns in some complex CALS tables, the stylesheets need to know the page width. The $nominal-page-width is used for this value.
