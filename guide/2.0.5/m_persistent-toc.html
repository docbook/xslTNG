m:persistent-toc

m:persistent-toc — Mode for generating the persistent table of contents.

Mode:
{http://docbook.org/ns/docbook/modes}persistent-toc
Defined in:
modules/toc.xsl (3)
Since:
2.0.0

Description

If $persistent-toc is true, the document will be processed in this mode to produce the contents of the persistent ToC. By default, this produces the same ToC as the m:toc mode.
