$image-property-warning

$image-property-warning — Warn if image properties cannot be obtained?

Parameter:
{}image-property-warning
Defined in:
param.xsl
Used in:
param.xsl, modules/objects.xsl
Used by:
f:object-properties()

Synopsis

  |$image-property-warning := true()

Description

This flag controls whether or not an xsl:message is emitted when the ext:image-properties() extension function is unavailable.
