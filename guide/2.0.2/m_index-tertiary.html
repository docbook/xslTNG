m:index-tertiary — Mode for generating index tertiary entries.

Mode:
{http://docbook.org/ns/docbook/modes}index-tertiary
Defined in:
modules/index.xsl

Description

When generating an index, each distinct tertiary indexterm is processed in m:index-tertiary mode to output its tertiary entry.
