$lists-of-procedures

$lists-of-procedures — Generate a list of figures?

Parameter:
{}lists-of-procedures
Defined in:
param.xsl
Used in:
param.xsl, modules/toc.xsl

Synopsis

  |$lists-of-procedures as xs:string := 'false'

Description

If true, a list of procedures will be generated. By default, they are only generated for books and sets.

This is a pseudo-boolean parameter. It is considered true if it has the value '1’, ‘yes’, or ‘true’. Any other value is considered false.
