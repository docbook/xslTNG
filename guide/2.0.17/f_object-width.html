f:object-width

f:object-width — Returns the width of a mediaobject.

Function:
{http://docbook.org/ns/docbook/functions}object-width#1
Defined in:
modules/objects.xsl
Used in:
modules/objects.xsl
Used by:
f:mediaobject-viewport(), f:object-scalefit()

Synopsis

f:object-width($info as map(*)) as map(*)

Description

This function returns the width of an object as a length (see Section 2, “Lengths and units”), if it has one. If $image-ignore-scaling is true, or no width is specified for the object, the empty length is returned.
