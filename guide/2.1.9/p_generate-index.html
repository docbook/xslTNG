$generate-index

$generate-index — Automatically generate an index?

Parameter:
{}generate-index
Defined in:
param.xsl
Used in:
param.xsl, modules/index.xsl

Synopsis

  |$generate-index := 'true'

Description

If a document contains an empty setindex, index, or indexdiv, and $generate-index is true, then an index will automatically be generated from any indexterms found in the document.
