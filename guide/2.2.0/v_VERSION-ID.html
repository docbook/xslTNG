$v:VERSION-ID

$v:VERSION-ID — A unique version identifier.

Variable:
{http://docbook.org/ns/docbook/variables}VERSION-ID
Defined in:
VERSION.xsl
Used in:
modules/head.xsl

Synopsis

$v:VERSION-ID := '8f2f63f'

Description

Generally speaking the $VERSION is sufficient to identify the stylesheets. The $VERSION-ID adds a unique identifier derived from the latest git commit.
