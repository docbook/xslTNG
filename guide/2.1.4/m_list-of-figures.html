m:list-of-figures

m:list-of-figures — Mode for generating a list of figures.

Mode:
{http://docbook.org/ns/docbook/modes}list-of-figures
Defined in:
modules/toc.xsl (2)

Description

If $lists-of-figures is true, then a list of figures will be generated. This mode is used to process the figure elements (if any) to produce that list.
