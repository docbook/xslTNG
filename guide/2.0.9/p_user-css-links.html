$user-css-links

$user-css-links — A list of user-defined CSS stylesheets.

Parameter:
{}user-css-links
Defined in:
param.xsl
Used in:
param.xsl, modules/variable.xsl

Synopsis

  |$user-css-links := ()

Description

This parameter is a space separated list of CSS stylesheets. Each will be added to the output after the standard stylesheets.
