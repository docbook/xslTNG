$err:DYNAMIC-PROFILE-SYNTAX-ERROR — Internal error processing dynamic profiles.

Variable:
{http://docbook.org/ns/docbook/errors}DYNAMIC-PROFILE-SYNTAX-ERROR
Defined in:
modules/errors.xsl
Used in:
Not used.

Synopsis

$dbe:DYNAMIC-PROFILE-SYNTAX-ERROR := xs:QName('dbe:DYNAMIC-PROFILE-SYNTAX-ERROR')

Description

This is an internal error. It indicates that the syntax of a dynamic profiling expression was unrecognized. Since dynamic profiling should only be applied to expressions that have been recognized, this error should never happen.
