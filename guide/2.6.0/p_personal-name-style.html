$personal-name-style — The style for formatting personal names.

Parameter:
{}personal-name-style
Defined in:
param.xsl
Used in:
main.xsl, param.xsl

Synopsis

$personal-name-style := ()

Description

The stylesheets can format personal names in a variety of ways. This is usually locale dependent. If $personal-name-style is specified, it wil be used in preferance to the locale dependent style. See t:person-name.
