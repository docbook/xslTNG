$err:INVALID-AREAREFS — Callout area refers to invalid target.

Variable:
{http://docbook.org/ns/docbook/errors}INVALID-AREAREFS
Defined in:
modules/errors.xsl
Used in:
main.xsl

Synopsis

$dbe:INVALID-AREAREFS := xs:QName('dbe:INVALID-AREAREFS')

Description

This error indicates that a calloutlist includes an ID/IDREF link to a callout, but the element identified by the ID is not an areaset, area, or callout.

This should be identified as a validation error by a Schematron rule.
