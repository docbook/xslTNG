f:chunk-filename — Returns the filename to use for a particular chunk.

Function:
{http://docbook.org/ns/docbook/functions}chunk-filename#1
Defined in:
modules/chunk.xsl
Used in:
modules/chunk.xsl
Used by:
f:chunk()

Synopsis

f:chunk-filename($node as element()) as xs:string

Description

When chunking, see Section 2.7, ““Chunked” output”, this function is called to determine the filename for a chunk. The default implementation considers relevant db processing instructions (filename, href, and basename pseudo-attributes) as well as the generated ID of the element.
