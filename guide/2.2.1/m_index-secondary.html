m:index-secondary — Mode for generating secondary index entries.

Mode:
{http://docbook.org/ns/docbook/modes}index-secondary
Defined in:
modules/index.xsl

Description

When generating an index, each distinct secondary indexterm is processed in m:index-secondary mode to output its secondary entry.
