$v:verbatim-syntax-highlight-languages

$v:verbatim-syntax-highlight-languages — Languages for which syntax highlighting should be performed.

Variable:
{http://docbook.org/ns/docbook/variables}verbatim-syntax-highlight-languages
Defined in:
modules/variable.xsl
Used in:
modules/head.xsl, modules/verbatim.xsl
Used by:
f:highlight-verbatim()

Synopsis

$v:verbatim-syntax-highlight-languages := tokenize($verbatim-syntax-highlight-languages, '\s+')

Description

The $verbatim-syntax-highlight-languages is a single string for the convenience of users. This variable contains the languages as a list.
