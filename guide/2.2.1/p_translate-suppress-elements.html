$translate-suppress-elements

$translate-suppress-elements — Elements in which translation is suppressed.

Parameter:
{}translate-suppress-elements
Defined in:
param.xsl
Used in:
param.xsl, modules/functions.xsl
Since:
2.1.3

Synopsis

  |$translate-suppress-elements := ''

Description

This parameter is a list of (local) element names. Any output for these elements will have the translate attribute with the value “no”.
